机缘

闲来无事，在工位上摸鱼。

从 阿里云瑶池数据库 微信公众号随手刷到一则 免费报名｜7月22日杭州城市行，解锁Apache Doris高效实时数据分析密码！ 的消息推文。既然 Apache 的活动都开到了家门口，没有不参加的理由，爽快的报了名。我肯定不会告诉你们，我连 Doris 是什么、怎么用都不知道。

赴会

杭州的夏季十分炎热，太阳毒辣的挂在天上。与我一行参会的还有我女友，她当天上午还要参加 华为2012实验室 举办的校招宣讲会。

Doris 线下会在 杭州电魂大厦 举办，于下午 14:00 开始。实际主办方是 SelectDB，是一家从事 Doris 开源和商业化的公司。参会人员不多，主要是大数据从业者。会上共有 4 位行业大佬做了业务落地的技术分享，包括来自 SelectDB、杭银消金、有赞和阿里云的技术专家。会议主要阐明了一个观点，推出 Doris 2.0，Doris 作为 ClickHouse 的下一代继任者。

会间休息，主办方提供了茶百道饮品，我和女友一人一杯，似乎我们是为了 2 杯奶茶而参加的会议，太没出息了，哈哈哈。

感想

在会上听取了技术大牛的技术分享，认识大数据系统架构的复杂性，从发现技术问题到新中间件解决问题，技术演变过程。作为一个普通的 Java 开发者，从优秀的开源项目中汲取养分，积蓄实力为开源项目贡献力量。

感谢主办方的奶茶，贵的确实好喝。

留一个小彩蛋，猜猜我在哪？杭州城市行已圆满结束，戳此获取演讲资料！
